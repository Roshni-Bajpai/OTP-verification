Setup instructions:

1. Download the zip folder or install the project using git.

2. Unzip it and open the project folder in code editor.

3. Right click the index.html file and select Open with Live Server.



How to try?:

1. Enter the email.

2. Open the console.

3. Click on 'Send OTP' button.

4. Copy the OTP from console.

5. Enter the OTP in the 4 blocks.

6. Click on Verify button.

7. Click on Logout button.
